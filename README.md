# 📊 Financial Company EDA & Machine Learning Project

## 🔍 Overview

This project performs end-to-end data analysis on global financial company data. It includes exploratory data analysis (EDA), SQL-based business insights, machine learning modeling, and dashboard visualization.

---

## 🎯 Objectives

* Analyze company performance using financial metrics
* Identify high-performing companies and countries
* Build predictive models for profit estimation
* Generate business insights for decision-making

---

## 📁 Dataset

* Source: Kaggle
* Contains financial data of top global companies
* Features include:

  * Sales
  * Profit
  * Assets
  * Market Value
  * Country

---

## 🧪 EDA & Insights

* Top companies by profit
* Sales vs Profit relationship
* Country-wise revenue distribution
* Asset efficiency analysis

---

## 🤖 Machine Learning

* Model: Regression model for profit prediction
* Features used:

  * Sales
  * Assets
  * Market Value
* Evaluation metrics:

  * R² Score
  * Mean Squared Error

---

## 🗄️ SQL Analysis

* Top 10 profitable companies
* Country-wise aggregation
* Profit margin analysis
* Business-driven queries

---

## 📊 Dashboard (Power BI)

* KPI cards (Revenue, Profit, Companies)
* Top companies visualization
* Sales vs Profit scatter plot
* Country-wise analysis
* Interactive filters
